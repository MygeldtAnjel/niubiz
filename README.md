# visanet(Perú) en Express

Implementacion de visanet en nodejs/express

Intalación
---

``` bash
npm install
```

Ejecución
---
cambiar los valores de las credenciales en config/settings.json 

``` json
{
  "visanet": {
    "user": "---",
    "password": "---",
    "merchantId": "---",
    "env": "dev"
  },
  "store": "..."
  
}
```

Iniciar el servidor

``` bash
npm start
```
